# 🧠 ASL Real-Time Sign Language Detection

## 📌 Overview
This project is a real-time American Sign Language (ASL) detection system using computer vision and machine learning.

## 🚀 Features
- Real-time hand tracking
- Data collection using webcam
- Model training
- Real-time prediction

## 🛠️ Tech Stack
- Python
- OpenCV
- MediaPipe
- Scikit-learn

## ▶️ Usage

### Collect Data
python asl_realtime.py collect

### Train Model
python asl_realtime.py train

### Predict
python asl_realtime.py predict

## ⚠️ Notes
- Requires webcam
- Accuracy depends on dataset
